# 类五笔输入法的码表转换

## 支持列表

**纯文本：**

> 纯文本当然都能输出

| 代号     | 词条格式                           | 描述 |
| :------- | :--------------------------------- | :--- |
| duoduo   | `词<tab>编码`                      | 多多 |
| bingling | `编码<tab>词`                      | 冰凌 |
| jidian   | `编码<tab>词1<space>词2<space>...` | 极点 |

**二进制：**

| 代号      | 格式   | 是否支持输出 | 描述               |
| :-------- | :----- | :----------: | :----------------- |
| baidu_def | `.def` |      是      | 百度手机自定义方案 |
| jidian_mb | `.mb`  |      否      | 极点               |
| fcitx4_mb | `.mb`  |      否      | fcitx4(WIP)        |
